## Rule Mining
This is a specialization of [Data Mining](Data_Mining) that deals with rules
- ''rules'' or ''patterns'' - correlation between features 

E.g. 
- like, lot $\Rightarrow$ positive
- not, like $\Rightarrow$ negative
