== ACID == 
ACID is an acronym that stands for 
- [*A*tomicity](Atomicity_%28databases%29)
- [*C*onsistency](Consistency_%28databases%29)
- [*I*solation](Isolation_%28databases%29)
- [*D*urability](Durability_%28databases%29)

ACID defines desirable properties for database transactions
